# Инструкция по использованию программы «Слабое звено»

Программно-аппаратный комплекс для проведения телевизионной игры «Слабое звено».

---

## 1. Общие сведения

Программа состоит из трёх основных частей:

| Компонент | Описание |
|-----------|----------|
| **Пульт оператора** | Главное окно — управление игрой, ответы, голосование |
| **Экран ведущего** | Окно для отображения на проекторе/телевизоре |
| **Редактор вопросов** | Отдельная программа для редактирования базы вопросов |

---

## 2. Запуск

### 2.1 Расположение файлов

После сборки файлы находятся в папке:

```
Build\Debug\net10.0-windows\
```

или

```
bin\Debug\net10.0-windows\
```

### 2.2 Запуск основного приложения

1. Запустите **WeakestLink.exe** — откроется **Пульт оператора**.
2. При необходимости запустите **WeakestLink.QuestionEditor.exe** — откроется **Редактор вопросов**.

### 2.3 Запуск экрана ведущего

1. На пульте оператора нажмите кнопку **HOST SCREEN** (или **ANALYTICS**).
2. Либо экран ведущего подключается по сети (см. раздел «Сеть»).

---

## 3. Пульт оператора (главное окно)

### 3.1 Подготовка к игре

1. **READY** — подготовка раунда (установка таймера, загрузка вопросов, определение стартового игрока).
2. **PLAY** — старт раунда. Таймер начинает отсчёт.

### 3.2 Во время раунда (игра)

| Кнопка | Действие |
|--------|----------|
| **ВЕРНО** | Правильный ответ — цепочка растёт, ход переходит следующему игроку |
| **ПАС** | Игрок пасует — цепочка сбрасывается, сумма «сгорает» |
| **НЕВЕРНО** | Неправильный ответ — цепочка сбрасывается, сумма «сгорает» |
| **БАНК** | Игрок забирает накопленную сумму в банк раунда |

### 3.3 Окончание раунда и голосование

1. Раунд завершается по таймеру или при достижении полного банка (50 000 ₽).
2. **VOTING START** — нажать, когда готовы начать голосование. Включается трек диктора, запускается таймер 29 секунд.
3. **VOTING END** — нажать по окончании голосования. Включается трек ведущего, запускается таймер 15 секунд.

### 3.4 Исключение игрока

1. В выпадающем списке **ИСКЛЮЧЕНИЕ** выберите имя игрока.
2. Нажмите **ИСКЛЮЧИТЬ**.
3. Нажмите **NEXT ROUND** для перехода к следующему раунду.

### 3.5 Специальные кнопки

| Кнопка | Назначение |
|--------|------------|
| **BEFORE VOTING** | Появляется после полного банка — нажать перед голосованием (унижения под музыку) |
| **ELIMINATION** | Переход к моменту исключения |
| **NEXT ROUND** | Подготовка нового раунда после исключения |
| **ПЕРЕЙТИ К ФИНАЛУ** | Появляется в предфинале (2 игрока) — переход к финальной дуэли |
| **RESET IDLE** | Сброс игры в начальное состояние |

### 3.6 Дополнительно

- **HOST SCREEN** — открыть экран ведущего (если он в том же процессе).
- **ANALYTICS** — открыть окно аналитики раунда (сильное/слабое звено, прогноз выбывания).
- **! ПАНИКА !** — экстренная остановка звука и сброс состояния.

---

## 4. Экран ведущего

- Отображает текущий вопрос и ответ (только ответ скрыт до нажатия).
- Показывает таймер, цепочку, банк.
- В режиме голосования — экран исключения.
- В финале — дуэль двух игроков с индикаторами верно/неверно.

---

## 5. Редактор вопросов

### 5.1 Переключение файлов

- В выпадающем списке **Файл** выберите:
  - **questions.json** — вопросы основных раундов
  - **final_questions.json** — вопросы финальной дуэли

### 5.2 Основные действия

| Кнопка | Действие |
|--------|----------|
| **Загрузить** | Перезагрузить вопросы из файла |
| **Сохранить** | Сохранить изменения в файл |
| **Добавить** | Добавить новый вопрос |
| **Удалить** | Удалить выбранный вопрос |
| **Очистить всё** | Удалить все вопросы (с подтверждением) |

### 5.3 Редактирование

1. Выберите вопрос в таблице.
2. Измените **Текст**, **Ответ** и **Раунд** в форме внизу.
3. Нажмите **Применить изменения**.

### 5.4 Поиск

Введите текст в поле поиска — список отфильтруется по совпадению с текстом вопроса или ответа.

---

## 6. Сеть

- **Пульт** — TCP-сервер на порту **8888**.
- **Экран ведущего** и другие клиенты подключаются по IP-адресу пульта.
- На пульте отображается QR-код и IP для подключения.
- Данные (вопросы, таймер, банк, голосование) синхронизируются автоматически.

---

## 7. Типичный сценарий игры

1. **READY** → подготовка раунда.
2. **PLAY** → старт раунда.
3. Во время раунда нажимать **ВЕРНО** / **ПАС** / **НЕВЕРНО** / **БАНК** в зависимости от ответа игрока.
4. По окончании времени или при полном банке — **VOTING START** (по готовности).
5. После таймера голосования — **VOTING END**.
6. Выбрать игрока в списке → **ИСКЛЮЧИТЬ**.
7. **NEXT ROUND** → повторять с шага 1.
8. При 2 игроках — **ПЕРЕЙТИ К ФИНАЛУ** → финальная дуэль.
9. По окончании дуэли — показан победитель.

---

## 8. Цепочка и банк

- **Цепочка:** 1000 → 2000 → 5000 → 10000 → 20000 → 30000 → 40000 → 50000 ₽
- **Верный ответ** — переход на следующий шаг.
- **Пас / Неверно** — цепочка обнуляется, сумма «сгорает».
- **Банк** — сумма с цепочки переносится в банк раунда.

- Windows 10/11
