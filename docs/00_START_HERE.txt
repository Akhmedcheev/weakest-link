🎯 ОСНОВНЫЕ ФУНКЦИИ API
════════════════════════════════════════════════════════════════════════════

screen.SetLevel(3)              // Перейти на уровень 3
screen.NextLevel()              // На следующий уровень
screen.PreviousLevel()          // На предыдущий уровень
screen.GetCurrentLevel()        // Получить текущий уровень
screen.SecondsRemaining = 30    // Установить время таймера (в секундах)
int time = screen.SecondsRemaining  // Получить оставшееся время
screen.Dispose()                // Остановить таймер и очистить

В файле BroadcastScreen.xaml.cs настраиваются эти константы:

private const double OverlapOffset = 25.0;
  ↓ Толщина нахлеста плашек в стопке
  ├─ ↑ Увеличьте для более разреженной стопки
  └─ ↓ Уменьшьте для плотнее упакованной стопки

private const double StandardSpacing = 50.0;
  ↓ Расстояние между плашками неиграемых уровней
  ├─ ↑ Увеличьте для более дальних плашек
  └─ ↓ Уменьшьте для более близких плашек

private const int AnimationDurationMs = 500;
  ↓ Длительность анимации падения плашки (миллисекунды)
  ├─ ↑ Увеличьте для медленной анимации
  └─ ↓ Уменьшьте для быстрой анимации

❓ ЧАСТЫЕ ВОПРОСЫ
════════════════════════════════════════════════════════════════════════════

Q: Как изменить количество уровней?
A: Отредактируйте массив amounts[] в InitializeChain()

Q: Как ускорить/замедлить анимацию?
A: Измените AnimationDurationMs (меньше = быстрее)

Q: Как сделать плашки плотнее в стопке?
A: Уменьшьте OverlapOffset (например, до 15 вместо 25)

Q: Что если изображения не загружаются?
A: 1. Проверьте путь в коде
   2. Проверьте Build Action = Resource в свойствах файла
   3. Убедитесь что файлы в папке Assets проекта

Q: Как использовать в существующем проекте?
A: Скопируйте XAML и CS файлы, добавьте изображения в Assets
