# Финальная система аналитики - "Слабое звено" v2.0

Один из главных документов (список всех — в [RULES.md](RULES.md)).

## Обзор
Создана комплексная система аналитики с тремя ключевыми метриками:
1. **Сильное/Слабое звено** - традиционный анализ с учетом пасов
2. **Прогноз выбывания** - умная система с защитой хороших игроков
3. **Перестраховщик** - новая метрика "трусости" банковых операций

## ✅ Исправленные проблемы

### 1. Математика слабого звена
- **Пасы = ошибки**: `TotalMistakes = IncorrectAnswers + Passes`
- **Сбалансированная логика**: ошибки → деньги → верные ответы → процент

### 2. Прогноз выбывания
- **Защита хороших игроков**: `CorrectAnswers > 0 && TotalMistakes == 0`
- **Сбалансированные веса**: слабое звено (2), ошибки (2), остальные (1)
- **Прозрачность**: объяснение причин выбора

### 3. Метрика "Перестраховщик"
- **Сбор данных**: `BankPressCount` увеличивается при каждом "БАНК!"
- **Расчет**: `AverageBankAmount = BankedMoney / BankPressCount`
- **Определение**: минимальный средний банк среди банкировавших

```csharp
public class PlayerRoundStats
{
    public int TotalMistakes => IncorrectAnswers + Passes;
    public double AverageBankAmount => BankPressCount > 0 ?
```

## 🧮 Алгоритмы работы

### 1. Слабое звено
```csharp
.OrderByDescending(p => p.TotalMistakes)    // Макс ошибки
.ThenBy(p => p.BankedMoney)                  // Минимум денег
.ThenBy(p => p.CorrectAnswers)               // Минимум верных
.ThenBy(p => p.SuccessPercentage)           // Худший процент
```

```csharp
// Защита хороших игроков
```

```csharp
var bankers = playerStats.Where(p => p.BankPressCount > 0);
var panicBanker = bankers.OrderBy(p => p.AverageBankAmount).First();
```

## 🎨 Визуальная индикация

### Цветовая схема:
- **🟢 Зеленый фон** - сильное звено
- **🔴 Красный фон** - слабое звено  
- **🟣 Фиолетовый фон** - главный перестраховщик
- **🟠 Оранжевая рамка** - прогноз выбывания

### Таблица статистики:
- **Ошибки** - выделены красным жирным
- **Средний банк** - розовый цвет, "---" если нет операций
- **Успешность** - "---" если не было вопросов

## 🔧 Преимущества системы

3. **Тактика**: выявляется стиль банкового поведения
4. **Прозрачность**: понятные объяснения решений
