# Метрика "Трусости" (Перестраховщик) - "Слабое звено"

Один из главных документов (список всех — в [RULES.md](RULES.md)).

## Обзор
Добавлена новая аналитическая метрика для определения игроков, которые слишком рано кладут мелкие суммы в банк, обрывая потенциально сильную цепь команды.

## Компоненты

### 1. GameEngine.cs - Расширение статистики

UpdatePlayerStat(CurrentPlayerTurn, s =>

### 2. StatsAnalyzer.cs - Новые метрики

#### Расширение `PlayerRoundStats`:
```csharp
public int BankPressCount { get; set; } // Количество банковских операций

// Средний размер банка (защита от деления на ноль)
public double AverageBankAmount => BankPressCount > 0 ? 
    Math.Round((double)BankedMoney / BankPressCount, 0) : 0;
```

#### Новое поле в `RoundAnalytics`:
```csharp
public string PanicBanker { get; set; } = ""; // Главный перестраховщик
```

#### Метод определения "Перестраховщика":
```csharp
private string DeterminePanicBanker(List<PlayerRoundStats> playerStats)
{
    // Только игроки, которые банкировали хотя бы раз
    var bankers = playerStats.Where(p => p.BankPressCount > 0).ToList();
    
    if (bankers.Count == 0) return ""; // Никто не банкировал
    
    // Игрок с наименьшим средним размером банка
    return bankers.OrderBy(p => p.AverageBankAmount).First().Name;
}
```

## Алгоритм работы

### 1. Сбор данных:
- Каждое нажатие кнопки "БАНК!" увеличивает `BankPressCount`
- Сумма денег добавляется в `BankedMoney`
- Данные привязаны к текущему игроку и раунду

### 2. Расчет метрик:
- **Средний банк** = `BankedMoney / BankPressCount`
- Защита от деления на ноль
- Округление до целых значений

### 3. Определение "Перестраховщика":
1. Фильтр: только игроки с `BankPressCount > 0`
2. Сортировка: по возрастанию `AverageBankAmount`
3. Выбор: игрок с минимальным средним банком

## Пример работы

### Сценарий:
- **Алексей**: 3 банка → 1000 + 1500 + 2000 = 4500 ₽ (средний: 1500 ₽)
- **Борис**: 2 банка → 500 + 800 = 1300 ₽ (средний: 650 ₽)  
- **Виктор**: 1 банк → 3000 ₽ (средний: 3000 ₽)

**Результат**: Борис - главный перестраховщик (минимальный средний банк: 650 ₽)

1. **Выявление тактики** - показывает игроков с осторожной стратегией
